# Nomoddiy Aktivlar (NA) Tizimi

Ushbu tizim tashkilotning nomoddiy aktivlarini (patentlar, litsenziyalar, dasturiy ta'minot va h.k.) hisobga olish va ularning amortizatsiyasini hisoblash uchun mo'ljallangan.

## Joylashuv
Asosiy fayl: `nomoddiy_aktivlar/index.html`

## Xususiyatlari
*   **0400** seriyali schyotlar bo'yicha aktivlarni kirim qilish.
*   Oylik amortizatsiya (eskirish) hisoblash (**0500** schyot).
*   Qoldiq qiymatni avtomatik hisoblash.
*   Operatsiyalar tarixi.
*   Ma'lumotlarni brauzer xotirasida (LocalStorage) saqlash (`na_assets`).
*   Chop etish (Print) imkoniyati.

## Qanday ishgatushiriladi?
`index.html` faylini istalgan zamonaviy brauzerda (Chrome, Edge, Firefox) oching. Internet talab qilinmaydi.
